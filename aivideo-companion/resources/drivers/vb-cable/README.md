Place the VB-CABLE Windows installer here before building the desktop app:

- `VBCABLE_Setup_x64.exe` (preferred)

Download from https://vb-audio.com/Cable/ (donationware).

Bundling VB-CABLE requires following VB-Audio's distribution terms:
https://vb-audio.com/Services/licensing.htm

The desktop setup wizard runs the installer with silent flags (`-h -i -H`).
Windows may still show a driver security prompt — users must approve it.
